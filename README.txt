📁 Project Structure:
---------------------
- index.html        -> Main webpage layout and content

1. Responsive personal portfolio layout
2. Left dashboard with internal navigation
3. Light/Dark mode toggle with smooth theme transition

5. Skill list with icons for HTML, CSS, JS, Python, etc.

7. Hometown photo section
8. Text Analyzer to count words, letters, etc.
9. Styled PDF CV download section
